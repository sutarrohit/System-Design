## **ğŸ”¹ Disadvantages of Horizontal Scaling**

-   If one server crashes, users should not lose data.

ğŸ”´ **Increased Load on Downstream Services**

5ï¸âƒ£ **Implement Caching**

## **ğŸ”¹ Conclusion**
